# Bud Challenge

## Usage

Navigate to the bud project folder:

```
run 'npm i'
```

```
then 'npm run build'
```

Navigate to ~bud/server/app.js 

```
run 'node app.js'
```

go to [localhost:8080](http://localhost:8080/) to see the app :)

## Testing

From the bud project folder:

```
'npm run test'
```
